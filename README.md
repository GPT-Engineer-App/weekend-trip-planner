# weekend-trip-planner

A hype-page for a weekend trip me and my friends are going on this autumn. I want an idea form that lets people input ideas for the trip, a form for entering music suggestions for a shared playlist, and a space for general info about the trip.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/weekend-trip-planner.git

```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
